# ChatSphere
# 💬 ChatSphere

A modern real-time communication platform inspired by Discord, Slack, and WhatsApp.

> 🚧 This project is currently under active development.

---

## 📌 Overview

ChatSphere is a full-stack real-time messaging application built using the MERN stack. It aims to provide secure, fast, and scalable communication with features such as private messaging, chat rooms, media sharing, online presence, and voice/video calling.
